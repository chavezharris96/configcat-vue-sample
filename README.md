# configcat-vue-sample

Sample app for the **configcat-vue** npm package:

**[https://github.com/codedbychavez/configcat-vue](https://github.com/codedbychavez/configcat-vue)**

## Project setup

1. Clone this repo

2. Install dependencies

```sh
npm install
```

3. Run the app

```sh
npm run dev
```
